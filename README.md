# Huma Qureshi - Portfolio

Welcome to the source code for my personal portfolio website!

## Overview
This is a modern, responsive portfolio showcasing my work in Full-Stack Development, Agentic AI, and Cyber Security.

## Deployment
This project is pre-configured and ready to be deployed to **Vercel** or **GitHub Pages**. 
It is a static site, meaning no complex build steps are required. 

- **Vercel**: Simply import the repository in Vercel. The `vercel.json` and `package.json` are already configured to deploy the root directory as a static site.
- **GitHub Pages**: You can enable GitHub pages from the repository settings, pointing it to the `main` branch root.

## Local Development
To run this locally, you can use any static server, for example:
```bash
npx serve .
# OR
python -m http.server 3000
```
